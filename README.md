# Olá! 👋 Eu sou o Rafael Yuri (ou só Rafael!)

**Engenheiro Químico | Pesquisador (Pós-doutorando) | Desenvolvedor C++ | Entusiasta Open Source**

Sou um apaixonado por ciência, tecnologia, programação e resolução de problemas complexos. Atualmente, atuo como pesquisador de pós-doutorado, desenvolvendo tecnologias para o setor de Petróleo e Gás (O&G). Meu foco de trabalho envolve a recuperação e o tratamento de fluidos de perfuração em peneiras vibratórias, combinando engenharia e computação de alta performance. 

E como um bom mineiro, adoro acompanhar tudo isso (especialmente o código) com um excelente café e boas conversas! ☕🧀
